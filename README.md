
# Web-Design-Project

Repository for the Web Design Module

This repository is a static website, designed for a Web Dev Solution Company called
Moo-Deng Web Solutions. It has a landing page, multiple form pages, an 'about us' page and a 'careers' page.

Features List

1. The website has mulitple forms, texts and images that work together to make it cohesive.
2. The website accomodates almost all viewports, through the implementation of CSS.
3. The website uses basic Java Script to alert users.

Future Improvements

1. Connecting the Website to a database for storing client communications and job applications.
2. Creating a display page for viewing and managing client communications and job applications.
3. Rendering job opportunities using a JSON file from a 'Vacancies' table in the database.
4. Implimenting AJAX where it is applicable on the site( mainly the display and careers pages).
5. Expanding the webpages to include 'demos' page: A page to showcase more website formats and  functionalities.
6. Testing the final website with a simulated back-end.
